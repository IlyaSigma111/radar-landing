# РАДАР Landing Page

Рекламный лендинг для РАДАР — мониторинг активности Движения Первых.

## Стек
- HTML5 + CSS3 + Vanilla JS
- Без зависимостей

## Деплой на GitHub Pages
1. Пушить в ветку `gh-pages`
2. Сайт доступен на: `https://[username].github.io/[repo]/`

## Локальный запуск
Просто откройте `index.html` в браузере.
